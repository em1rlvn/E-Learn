## 🚀 Развёртывание (Production)

### Вариант 1: Сборка локально и публикация на GitHub Pages

```bash
# 1. Установить зависимости
npm install

# 2. Создать сборку
npm run build

# 3. Готовые файлы будут в папке /build
```

### Вариант 2: Публикация на Vercel (рекомендуется)

```bash
# 1. Установить Vercel CLI
npm install -g vercel

# 2. Залогиниться
vercel login

# 3. Развернуть проект
vercel

# 4. Следовать указаниям в командной строке
```

### Вариант 3: Публикация на Netlify

```bash
# 1. Создать сборку
npm run build

# 2. Установить Netlify CLI
npm install -g netlify-cli

# 3. Развернуть
netlify deploy --prod --dir=build
```

### Вариант 4: Публикация на Heroku (если есть бэкенд)

```bash
# 1. Создать Procfile в корне проекта
echo "web: npm run start" > Procfile

# 2. Создать сборку
npm run build

# 3. Инициализировать Git репозиторий
git init
git add .
git commit -m "Initial commit"

# 4. Создать приложение на Heroku
heroku create your-app-name

# 5. Развернуть
git push heroku main
```

## 📝 Как использовать приложение

### Для тестирования аутентификации:

**Регистрация:**
1. Перейти на страницу Register
2. Заполнить имя, email, пароль
3. Нажать Register
4. На странице Verify ввести любой OTP (система mock)
5. Перенаправлено на Login

**Вход:**
1. Использовать только что зарегистрированные данные
2. Или попробовать другую пару email/password (в mock системе любые подойдут после регистрации)

**Выход:**
1. Нажать кнопку "Logout" в header

### Для изучения курсов:

1. Перейти на Courses
2. Использовать поиск, фильтры, сортировку
3. Выбрать курс и кликнуть "Get Started"
4. Просмотреть видео лекции
